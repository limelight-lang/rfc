# rfc

RFC documents for the Limelight project, a Runtime designed for PHP.

Key principle: [attributes.md](attributes.md) — Limelight adds no new
syntax to PHP; every capability enters through native attributes, and
the compiler's whole-program analysis materializes its findings back
into code as the same attributes.

Deferred and upcoming work is tracked in [BACKLOG.md](BACKLOG.md).
